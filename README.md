# Traffic K9 Counter — счётчик транспорта на перекрёстке

Кроссплатформенное приложение для ручного учёта транспортных средств на перекрёстке с гибкой настройкой направлений, экспортом в Excel и аналитикой.

## Ссылка на репозиторий
https://github.com/Kango911/traffic_k9_counter.git

## Возможности

- Выбор **въездов** и **выездов** перед стартом (N, S, E, W).
- 10 типов ТС: легковые, микроавтобусы, средние и большие автобусы, малые/средние/тяжёлые грузовики, автопоезда, троллейбусы, трамваи.
- Кнопки-счётчики: **левая кнопка** +1, **правая кнопка** –1.
- Порядок направлений: **направо → прямо → налево → разворот**.
- Всплывающие описания типов ТС (кнопка `i`).
- Экспорт в Excel с:
  - полной таблицей по направлениям и типам;
  - общим количеством ТС;
  - количеством и долей ТС без общественного транспорта;
  - долями поворотов для каждого въезда.
- Изменяемый размер окна, адаптивная вёрстка, прокрутка.
- Иконка приложения "K9".

## Технологии

- Python 3.8+
- PySide6 (Qt for Python)
- openpyxl (экспорт в Excel)

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/Kango911/traffic_k9_counter.git
cd traffic_k9_counter

# Создание виртуального окружения (рекомендуется)
python -m venv venv
source venv/bin/activate   # Linux/macOS
venv\Scripts\activate      # Windows

# Установка зависимостей
pip install PySide6 openpyxl

# Запуск
python traffic_counter.py
```

## Как пользоваться

1. При запуске выберите въезды (откуда едут) и выезды (куда поворачивают). Нажмите OK.
2. В главном окне введите название перекрёстка и дату записи.
3. Для каждого направления и типа ТС нажимайте на кнопку:
   - **Левая кнопка** — увеличить счётчик.
   - **Правая кнопка** — уменьшить (до 0).
4. Нажмите **«Экспорт в Excel»**, выберите имя файла — получите полный отчёт.

## Пример отчёта в Excel

- Лист «Перекрёсток» содержит:
  - шапку с названием перекрёстка и датой;
  - таблицу направлений × типы ТС;
  - итоги: общее количество ТС, количество без общественного, доля;
  - таблицу долей поворотов по каждому въезду.
